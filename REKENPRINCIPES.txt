DEEL 1: HET KERNPRINCIPE
================================================================================

De tool gebruikt GEEN volledige NTA8800 berekening (die is 1120 pagina's en
te complex voor een snelle inschatting).

In plaats daarvan werken we met een STATISTISCH MODEL:

    EP2 = Basistabel(bouwjaar, woningtype) + Som(correctiefactoren)

Dit model is afgeleid uit analyse van ECHTE energielabels:
- 359 woningen uit uniec3 bestanden (software voor EP-adviseurs)
- 5,7 miljoen labels uit EP-Online totaalbestand

Het principe is hetzelfde als hoe een ervaren adviseur ter plekke een
inschatting maakt: op basis van ervaring en patronen, niet op basis van
alle 1120 pagina's formules.

================================================================================

2.1 UNIEC3 DATABASE (primaire bron voor correctiefactoren)
----------------------------------------------------------
Bron: 359 geanalyseerde energielabel berekeningen uit COMPLETED map
Formaat: ZIP-bestanden met entities.json (woning kenmerken + EP2)
Geextraheerd: december 2025

Wat we hieruit hebben gehaald:
- Gemiddelde EP2 per type verwarming, ventilatie, afgifte
- Impact van elke maatregel t.o.v. baseline
- Bouwjaar per woning (veld: GEB_BWJR)
- Validatie van labelgrenzen

2.2 EP-ONLINE TOTAALBESTAND (secundaire bron)
---------------------------------------------
Bron: Officieel totaalbestand van alle energielabels in Nederland
Download: 3 januari 2026
Omvang: 5,7 miljoen labels (1,4 GB CSV)
Opgeslagen: SQLite database voor snelle queries

Wat we hieruit halen:
- Buurt-vergelijking (alle labels in dezelfde postcode)
- EP2 waarden van officieel geregistreerde labels
- Bouwjaar, energieklasse per adres

2.3 BAG (Basisregistratie Adressen en Gebouwen)
-----------------------------------------------
Bron: PDOK (Publieke Dienstverlening Op de Kaart)
API: https://api.pdok.nl/bzk/locatieserver/search/v3_1/
WFS: https://service.pdok.nl/lv/bag/wfs/v2_0

Wat we hieruit halen:
- Bouwjaar (authoratieve bron, Kadaster)
- Gebruiksoppervlakte in m2
- Adres verificatie

2.4 BOUWBESLUIT (historisch)
----------------------------
Bron: handelbouwadvies.nl, geregeld24.nl, RVO.nl
Wat we hieruit halen:

DEEL 3: DE BASISTABEL (Bouwjaar x Woningtype)
================================================================================

3.1 WAAROM BOUWJAAR?
--------------------
Het Bouwbesluit stelt minimale isolatie-eisen. Deze zijn door de jaren heen
aangescherpt. Bouwjaar is daarom een PROXY voor het isolatieniveau:

| Periode     | Rc gevel | Rc dak | Rc vloer | Toelichting               |
|-------------|----------|--------|----------|---------------------------|
| Voor 1975   | 0        | 0      | 0        | Geen isolatie-eisen       |
| 1975-1982   | ~1.0     | ~1.0   | 0        | Eerste eisen, matig       |
| 1983-1991   | 1.30     | 1.30   | 0.52     | Bouwbesluit 1983          |
| 1992-2011   | 2.50     | 2.50   | 2.50     | GROTE SPRONG! Goede iso   |
| 2012-2014   | 3.50     | 3.50   | 3.50     | Bouwbesluit 2012          |
| 2015-2020   | 4.50     | 6.00   | 3.50     | Aanscherping              |
| 2021+       | 4.70     | 6.30   | 3.70     | BENG (bijna energieneutr) |

3.2 WAAROM WONINGTYPE?
----------------------
Woningtype bepaalt de COMPACTHEID (Als/Ag ratio):
- Als = verliesoppervlak (gevels, dak, vloer naar buiten)
- Ag = gebruiksoppervlakte (vloeroppervlak)

Hoe meer buitengevel per m2 vloer, hoe meer warmteverlies.

| Woningtype              | Als/Ag    | Compactheid    |
|-------------------------|-----------|----------------|
| Appartement (midden)    | 0.8 - 1.2 | Zeer gunstig   |
| Appartement (hoek/boven)| 1.2 - 1.8 | Gunstig        |
| Tussenwoning            | 1.2 - 2.0 | Gunstig        |
| Hoekwoning              | 1.6 - 2.4 | Matig          |
| 2-onder-1-kap           | 1.6 - 2.4 | Matig          |
| Vrijstaand              | 1.8 - 3.0 | Ongunstig      |

De compactheid zit IMPLICIET in de basistabel: vrijstaand heeft hogere
basiswaarde dan appartement.

4.5 ZONNEPANELEN (NTA8800 FORFAITAIR)
-------------------------------------
Formule:
    Correctie = -(panelen x 289 x 0.9 x 1.45) / m2 vloeroppervlak

Waarbij:
- 289 Wp = vermogen per paneel (175 Wp/m2 x 1.65m2, monokristallijn 2018+)
- 0.9 = performance ratio (NTA8800 forfaitair)
- 1.45 = primaire energiefactor elektriciteit

Voorbeeld 100m2 woning:
| Panelen | Opbrengst  | Correctie |
|---------|------------|-----------|
| 5       | ~1575 kWh  | -19       |
| 10      | ~3150 kWh  | -38       |
| 15      | ~4725 kWh  | -57       |

BELANGRIJK: Impact hangt af van woonoppervlakte!
Kleine woning = meer effect per paneel (delen door minder m2)
Grote woning = minder effect per paneel (delen door meer m2)

DEEL 6: WAT DE TOOL NIET DOET (VEREENVOUDIGINGEN)
================================================================================

De tool vraagt NIET naar:
- Exacte glasoppervlakte (m2)
- Exacte Rc-waarde van isolatie
- Type warmtepomp (lucht/water/bodem)
- Exacte U-waarde van kozijnen
- Orientatie van de woning (zon)
- Koudebruggen
- Infiltratieverlies (luchtdichtheid)
- Specifieke geometrie (dakkapellen, erkers)

Dit is BEWUST: het doel is 80% dekking met 20% van de input.
Een ervaren adviseur maakt ter plekke ook zo'n snelle inschatting.

7.2 BETROUWBAARHEIDSSCORE
-------------------------
De tool geeft een betrouwbaarheidsscore (0-100%):

| Factor                 | Bijdrage |
|------------------------|----------|
| Bouwjaar bekend (BAG)  | +20%     |
| m2 bekend (BAG)        | +15%     |
| Woningtype bekend      | +15%     |
| Verwarming ingevuld    | +15%     |
| Beglazing ingevuld     | +10%     |
| Zonnepanelen ingevuld  | +10%     |
| Geavanceerde modus     | +15%     |

Typische betrouwbaarheid:
- Alleen basismodus: 60-80%
- Geavanceerde modus: 75-95%
- Met BAG data: +10%

7.3 BEKENDE BEPERKINGEN
-----------------------
1. Pre-1992 woningen: grotere onzekerheid (isolatie kan sterk varieren)
2. Combinatie-effecten: niet alle interacties zijn gekwantificeerd
3. Kleine sample voor sommige types (bijv. elektrisch verwarmen: N=4)
4. Renovaties: niet altijd te detecteren uit bouwjaar alleen

DEEL 8: PRAKTIJKINZICHTEN ADVISEUR
================================================================================

Deze kennis komt van M. Rondema, ervaren EP-adviseur, en staat NIET in
de NTA8800.

8.1 HOOGSTE IMPACT (prioriteren)
--------------------------------
1. Warmtepomp: -110 kWh/m2 (ENORM)
2. Zonnepanelen: afhankelijk van m2 (GROOT)
3. WTW ventilatie: -60 kWh/m2
4. HR++ glas: -20 kWh/m2
5. Dakisolatie: -25 kWh/m2

8.2 LAGE IMPACT (niet prioriteren voor label)
---------------------------------------------
1. Vloerisolatie: -5 kWh/m2 (alleen comfort!)
2. Buitendeuren: minimaal
3. Oude mechanische ventilatie: soms slechter dan natuurlijk

8.3 VENTILATIE TRUC
-------------------
Als NIET op 3 plekken afzuigventiel aanwezig (keuken, badkamer, toilet):
-> Kies NATUURLIJK in de berekening
Dit is vaak gunstiger dan oude mechanische ventilatie met wisselstroom-
motor (energieverslinder).

8.4 HYBRIDE WARMTEPOMP
----------------------
Goedkoper dan volledig elektrisch, bijna zelfde EP2 resultaat.
Makkelijker te installeren. Aanrader voor verduurzaming.

================================================================================
DEEL 11: DISCLAIMER
================================================================================

Dit is een INSCHATTING op basis van statistische patronen uit echte
energielabels. Het is GEEN officieel energielabel.

Het werkelijke energielabel kan alleen worden vastgesteld door een
gecertificeerd energieadviseur met een volledige NTA 8800 berekening.

De tool geeft een indicatie die in ~80-90% van de gevallen binnen 1
labelstap van het werkelijke label zit, maar kan afwijken bij:
- Sterk gerenoveerde woningen
- Atypische woninggeometrie
- Combinaties van factoren die niet in de database voorkomen
- Woningen met bijzondere installaties
